# Comparador de algoritmos de ordenamiento

La finalidad de este proyecto es hacer un programa que importe listas en formato de archivo de texto, y ordene los datos utilizando en tiempo real 5 algoritmos de ordenamiento los cuales serán:

* Bubble sort.     Big O(n)2.
* Insertion sort.  n2
* Merge sort.      Big O n log n
* Quick sort.
* Heap sort.

El programa no necesita exportar las listas ordenadas, sin embargo, será algo que intentaré implementar. Lo que si necesita hacer el programa es llevar la cuenta de los pasos que utilizó cada algoritmo de ordenamiento y el tiempo que se tardó en ordenar la lista.

Guía personal:

* Hacer una interfáz gráfica.
* Implementar la función de importar archivos de texto y su contenido.
* Implementar los cinco algoritmos de ordenamiento.
* Implementar la función de ordenamiento de datos usando los cinco algoritmos.
