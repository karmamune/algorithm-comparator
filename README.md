# Comparador de algoritmos de ordenamiento

La finalidad de este proyecto es hacer un programa que genere listas de datos, y ordene los datos utilizando en de forma secuencial 5 algoritmos de ordenamiento los cuales serán:

* **Bubble sort**     $O(n^2)$
* **Insertion sort**  $O(n^2)$
* **Merge sort**      $O(n \log n)$
* **Quick sort**      $O(n^2)$
* **Heap sort**       $O(n \log n)$

El programa lleva la cuenta de los movimientos que utilizó cada algoritmo de ordenamiento y el tiempo que se tardó en ordenar la lista.

Los algoritmos están basados en los videos del canal de YouTube: **Coding with John**
